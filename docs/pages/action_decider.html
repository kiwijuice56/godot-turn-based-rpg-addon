ActionDecider

extends Node

ActionDeciders are nodes that use the context of a battle to choose an Action for a Fighter. In a node tree, they should usually be the children of Fighters and should contain only Actions as children. The base class does not make any actual decisions, as it is intended to be extended. Extension classes should each be one type of decision-maker, such as a player ActionDecider that reads player input or an enemy ActionDecider that chooses Actions at random.

decide(context: Dictionary) -> Dictionary

Chooses an Action child for the parent Fighter to undergo. Returns a diciontary containing the Action reference and array of Fighter targets. In your extension class, override this method and return any other information necessary for the partent Fighter to undergo their turn.
