Action

extends Node

Represent single things to be done by a Fighter in their turn, such as attacks or using items. They are intended to be extended and turned into modular scenes that do one singular thing in combat via their action function, which is called by parent Fighter nodes.

Variables

save_id: String

Signals

action_completed

Emitted by action function upon completion

Functions

action(user: Node, targets: Array) -> void

Called by Fighter to affect the targets and/or the user. Emits a signal with a dictionary consisting of the report of the action, which is captured by the user Fighter. In your extension class, override this method and implement things such as damage calculation or graphical effects.

Example Implementation:

func action(user: Node, targets: Array) -> void:
    # Fake variables for example only
    for target in targets:
        target.hp -= user.attack + power

    emit_signal("action_completed", {"Missed": false, "Critical": true, "Hit Weakness": true})
